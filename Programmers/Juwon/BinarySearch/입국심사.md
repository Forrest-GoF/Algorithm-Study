[입국심사](https://programmers.co.kr/learn/courses/30/lessons/43162?language=java)

```java
import java.io.*;
import java.util.*;

class Solution {
	public long solution(int n, int[] times) {
		long answer = 0;
		long left = 1;
		long right = 1000000000000000000L;
		long mid;
		
		while (left <= right){
			mid = (left + right) / 2;
			long maxPeople = 0;
			
			for(int time : times){
				maxPeople += mid / time;
			}
			
			if(maxPeople >= n){
				answer = mid;
				right = mid - 1;
			}else{
				left = mid + 1;
			}
		}
		
		return answer;
	}
}
```

### 모범 답안  
- 시간복잡도  
    - O(MlogN)(N: 입국심사를 기다리는 사람의 수, M: 심사관의 수)
- 공간복잡도  
    - O(M)
