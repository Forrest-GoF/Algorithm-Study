```java
import java.io.*;
import java.util.*;

class Solution {
	int answer;
	int copyOfTarget;
	int[] copyOfNumbers;

	void DFS(int L, int sum){
		if(L == copyOfNumbers.length){
			if(sum == copyOfTarget){
				++answer;
			}
		}
		else{
			DFS(L+1, sum + copyOfNumbers[L]);
			DFS(L+1, sum - copyOfNumbers[L]);
		}
	}

	public int solution(int[] numbers, int target) {
		copyOfTarget = target;
		copyOfNumbers = numbers;
		DFS(0, 0);

		return answer;
	}
}
```

### 모범 답안  
- 시간복잡도  
    - O(2^N)
- 공간복잡도  
    - O(N)
